##  Si algo no funciona:

1. **Verifica que Node.js esté instalado:**
   ```bash
   node --version
   npm --version
   ```

2. **Limpia e reinstala dependencias:**
   ```bash
   cd C:\Users\Usuario\Documents\papus-final-react
   npm install
   ```

3. **Intenta en otro puerto:**
   ```bash
   npm start -- --port 3001
   ```
